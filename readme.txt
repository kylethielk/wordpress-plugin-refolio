== Description ==

reFolio is a plugin portfolio providing elegant portfolios anywhere on your wordpress site! Define an unlimited number of 
portfolios which can be used anywhere on your wordpress site that shortcodes are supported. Easy to use and 100% wordpress integrated
means a quick snappy portfolio that lets your work shine.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `refolio` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look for the reFolio menu in your admin control panel to get started creating your portfolios.

== Frequently Asked Questions ==

= Is there a limit to the amount of portfolios I can have? =

Nope. No limit.

= Does this work on all browsers? =

Works perfectly and tested on all modern browsers. Requires javascript support.
